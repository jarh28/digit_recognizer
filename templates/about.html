Digit Recognizer

This website was develop by Luis A. Denis Miranda. I am an informatics engineer currently working at the Technological University of Havana.

I am an avid reader and an AI enthusiast. This project was design to showcase the potential applications of the latest techniques in Machine Learning. In this case, a neural network was trained to identify digits drawn by people. I employed the famous MNIST data used by Kaggle in the introductory computer vision competition.


How does it work?

When the user draws an digit in the canvas and click in the identify button, the image is send to the server to process. First, all the unnecessary white space which is around the digit is deleted. Then, the image is treated to fit in a 28x28 frame. This resulting image is fed into the trained neural network and its output is shown to the user.
